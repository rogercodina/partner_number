INTRODUCTION
------------

This Drupal 7 module assigns an autoincremental partner number to users. It
implements two different counters depending on user role. You can adapt the
source code to your needs.

 * You can change the next partner number to be assigned.

INSTALLATION
------------

 * Install as you would normally install a contributed Drupal module. See:
   https://drupal.org/documentation/install/modules-themes/modules-7
   for further information.

CONFIGURATION
-------------

 * Configure module's options in Administration » Configuration » People
   » Partner number configuration:

    - Alumni bold partner number: next number to be assigned.

    - Enterprise partner number: next number to be assigned.
